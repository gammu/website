{% trans "Python bindings for Gammu" %}

{% blocktrans %} Gammu also includes Python bindings which allow you to use (most) of Gammu functionality from Python language. {% endblocktrans %}

{% include "shortnews.html" %}

{% trans "Download" %}

{% blocktrans %} Python module is now standalone, so please download it separately. If you want to use binary package, the daemon might be included in separate binary package in your distribution, try looking for something like python-gammu. {% endblocktrans %}

{% trans "The easiest way to install it is using pip:" %}

pip install python-gammu

{% trans "Example" %}

{% blocktrans %} Using python-gammu is quite simple, for example sending of the text message can be done by following snippet: {% endblocktrans %} 

import gammu

sm = gammu.StateMachine()
sm.ReadConfig()
sm.Init()

message = {
    'Text': 'python-gammu testing message', 
    'SMSC': {'Location': 1},
    'Number': '+420800123465',
}

sm.SendSMS(message)
