Wammu

{% blocktrans %} Wammu is mobile phone manager running on Linux, Windows and possibly other platforms, where libGammu and wxPython works. The communication is made by Gammu library. Currently supported features: {% endblocktrans %}

  • {% blocktrans %}complete support (can read/edit/delete/copy) for contacts, todo, calendar{% endblocktrans %}
  • {% blocktrans %}can read/create/save/send/backup smses{% endblocktrans %}
  • {% blocktrans %}sending files to phone (OBEX and Sony Ericsson phones only){% endblocktrans %}
  • {% blocktrans %}sms composer for multi part smses (currently only text and predefined bitmap/sound can be edited){% endblocktrans %}
  • {% blocktrans %}display message including pictures and ringtones playback{% endblocktrans %}
  • {% blocktrans %}support for backup and import in various formats (vCard, vCalendar, vTodo, iCalendar, gammu own backup,â€¦){% endblocktrans %}
  • {% blocktrans %}export messages to mail (IMAP4, maildir and mailbox storages are supported){% endblocktrans %}
  • {% blocktrans %}searching for phone{% endblocktrans %}
  • {% blocktrans %}translated into several languages{% endblocktrans %}
  • {% blocktrans %}rated as best on many software servers{% endblocktrans %}
